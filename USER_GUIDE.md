# ScriptZero 新手使用指南

## 什么是 ScriptZero？

ScriptZero 是一个帮助您自动运行游戏脚本的工具，特别适用于原神（Genshin Impact）的日常任务自动化。它就像一个智能助手，可以代替您完成游戏中的一些重复性操作。

## 准备工作

### 1. 确保系统要求
- Windows 10 或更高版本的操作系统
- 已安装 Python 3.7 或更高版本
- 已安装原神游戏
- 已安装 BetterGI 自动化脚本框架

### 2. 下载和安装
1. 将 ScriptZero 项目文件夹保存到您的电脑上
2. 确保所有文件都在同一个文件夹中

## 第一步：创建配置文件

配置文件告诉 ScriptZero 您的游戏在哪里以及如何运行自动化任务。

### 方法一：使用配置向导（推荐）
1. 打开命令提示符（在项目文件夹地址栏输入 `cmd` 并按回车）
2. 输入以下命令并按回车：
   ```
   python -m src.cli config wizard
   ```
3. 按照提示操作：
   - 选择游戏（例如：选择"1"代表原神）
   - 输入原神游戏路径（例如：`F:\Genshin Impact\Genshin Impact Game\YuanShen.exe`）
   - 输入 BetterGI 路径（例如：`F:\better\BetterGI.exe`）
   - 按需设置其他参数

完成后会生成一个 `config.yaml` 文件。

## 第二步：运行自动化

### 方法一：命令行运行
在命令提示符中输入：
```
python -m src.cli start config.yaml
```

### 方法二：图形界面运行
双击运行 `start_gui.py` 文件，会打开图形界面。

## 重要注意事项

⚠️ **使用风险提示**：
1. 请确保您了解使用自动化工具可能违反游戏服务条款的风险
2. 合理使用，避免过度依赖
3. 保护好您的游戏账户安全

⚠️ **技术注意事项**：
1. 确保游戏和脚本框架路径正确
2. 保持游戏界面在前台，不要最小化
3. 确保分辨率和游戏设置不会影响图像识别

## 常见问题解答

### Q: 启动时提示找不到文件怎么办？
A: 检查配置文件中的游戏路径和脚本框架路径是否正确，确保文件真实存在。

### Q: 自动化无法正常工作怎么办？
A: 检查以下几点：
- 游戏是否已启动并运行
- BetterGI 是否已启动并运行
- 游戏窗口是否在前台
- 确保没有遮挡游戏界面的其他窗口

### Q: 如何停止自动化任务？
A: 关闭命令行窗口或点击图形界面的停止按钮。

## 高级功能

### 图像识别
ScriptZero 使用图像识别技术来检测游戏界面元素，这意味着它可以适应一些界面变化。

### 工作流程定制
通过修改配置文件，您可以定制自动化任务的具体步骤。

## 故障排除

如果遇到问题，请尝试以下步骤：
1. 检查所有路径是否正确
2. 确保依赖库已正确安装（运行 `pip install -r requirements.txt`）
3. 查看错误信息并根据提示解决问题
4. 确保游戏和脚本框架都可以手动正常运行
